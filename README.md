# 2dGameFrameworkv2



Hele APIet baserer seg på at man lager en scene. For å lage en Scene må man først lage, bakgrunn, en liste med wolr object, en spiller, en liste med enemies, et world object som er målet.


Får å lage listene anbefales det at man lager fiender og objekter hver for seg og kopiere de og leggger de inn i listene. Dette er for å ha ‘’Prefabs’’ klare for senere baner.
